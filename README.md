# Onboarding-Offboarding-automation
This project is designed to automate the process of onboarding and offboarding for Companies

Here will be multiple files to check, update, create, move, add, remove, disable and enable accounts in Active Directory

With the scripts on this project you will be able to set descriptions for users, update accounts, memberships, office locations, phone numbers, emails and more within the Windows Server and Active Directory services.

The project is fully based on Powershell, since is a Native scripting language for Windows Systems
